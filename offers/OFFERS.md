# Offers (v1 — Evidence-Backed)

## 1) AI Stack Rescue (2h)
**Price:** 149€

**Best for:** broken local/self-hosted AI stacks that need fast recovery and clear proof.

**Includes:**
- Incident triage and likely root-cause map
- Targeted repair actions inside agreed scope
- Before/after verification output
- Changed components list
- Rollback notes

---

## 2) Local AI Setup + TTS (1 day)
**Price:** 449€

**Best for:** OpenWebUI/Ollama/local-TTS setup that must be stable and reproducible.

**Includes:**
- Service integration (OpenWebUI + model runtime + TTS endpoint)
- Health/discovery/speech verification
- Drift guard setup (when suitable)
- Ops runbook and restart/recovery notes

---

## 3) Ops Audit + Launch Gate
**Price:** 299€

**Best for:** noisy systems that need a pass/warn/fail operational gate before presentation/deploy.

**Includes:**
- Core vs non-core failure triage
- Repair where sensible, disable/defer where appropriate
- Explicit rationale + re-enable path
- Launch gate verification snapshot

---

## 4) Monitoring & Alerting Baseline (0.5 day)
**Price:** 219€

**Best for:** teams that want visibility into service health without building a full SRE program.

**Includes:**
- baseline review of running monitoring components
- minimal dashboard/probe sanity check
- monitoring snapshot evidence export
- prioritized recommendations (top 3)

---

Read-only scope: Monitoring package does not include runtime write/config changes unless separately agreed.
