# Dark Clean Visual Style Guide (v1)

## Palette
- Background: `#0B0F14`
- Card: `#121A23`
- Primary text: `#EAF2FF`
- Secondary text: `#A9B8CC`
- Accent: `#6EE7FF`
- Success: `#5EE6A8`
- Warning: `#FFC46B`

## Typography
- Headline: Inter / SF Pro Display, Semibold, 38–46px
- Subhead: Inter Medium, 22–28px
- Body: Inter Regular, 16–20px
- Label/Tag: Inter Medium, 13–15px, uppercase tracking +4

## Card layout
1. Top label (SERVICE)
2. Big title
3. One-line promise
4. 3 bullets (what you get)
5. Price + duration row
6. Proof line (evidence ref)
7. CTA line (pilot/waitlist)

## Visual rules
- Keep one accent color per card
- Max 55 words body text per card
- Use high contrast, no gradients required
